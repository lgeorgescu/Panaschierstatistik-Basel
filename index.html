Rohdaten

Die Daten stammen von der Fachstelle für OGD Basel-Stadt und können in Rohform hier heruntergeladen werden. Für die Darstellungen wurden die Daten aufbereitet.

Das verwendete Titelbild stammt von der Website des Datenportals Basel-Stadt.

Entwicklung der Website

Das Gerüst der Website wurde vom Open-Data-Projekt von Lukas Oester und Jan Liechti übernommen. Ihre App findet sich unter folgendem Link.

Der gesamte Code befindet sich frei zugänglich auf einer Github-Repository.

Für die Entwicklung dieser Website wurden folgenden Programmiersprachen und Softwarepakete verwendet:

  • JavaScript für die Apps
  • D3.js v3 für das Sankey-Diagramm
  • Bootstrap v4.4.1
  • Sankey.js (Sankey inventor, basecode)
