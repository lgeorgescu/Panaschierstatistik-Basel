Daten- und Darstellungsauswahl


Zur einfacheren Lesbarkeit werden in der Beschreibung stets die Begriffe "Nodes" und "Links" verwendet.

  • Nodes: Die Boxen bzw. Knoten an beiden Enden des Diagramms
  • Links: Die Verbindungsstücke zwischen je zwei Nodes

Jeder Node auf der linken Seite des Diagramms steht für einen Stimmenspender. Der Empfänger auf der rechten Seite ist über einen Link mit dem Spender verbunden. Wenn man mit dem Mauszeiger auf einem Link verharrt, so wird der ensprechende Betrag an Stimmen angezeigt.

Der mit "Ohne" beschriftene Node steht in allen Darstellungen für die freie Liste.

Durch Klicken auf den entsprechenden Kreis kann aus den folgenden Datensätzen ausgewählt werden:


Alle Parteien
Der Datensatz umfasst die Kandidatenstimmen veränderter Wahllisten über alle Parteien und Wahlkreise hinweg.
Durch Klicken des Buttons "Fluss innerhalb einer Partein anzeigen" können die Stimmen, welche eine Partei sich selber gespendet hat, ausgeblendet werden.

Lesebeispiele
Beispiel 1

Über die Hälfte der Stimmen, welche die Partei GB spendet, geht an die eigene Partei.

Beispiel 2

Der grösste Teil der Stimmen, die die Partei GB an andere Parteien spendet, geht an die SP.

Einzelne Parteien
Im Dropdown-Menu kann eine Partei ausgewählt werden. Diese kann als Stimmen-Spender (Partei auf der linken Achse des Diagramms) oder als Stimmen-Empfänger (auf der rechten Achse) angezeigt werden.
Wahlkreise
In einem Dropdown-Menu kann der gewünschte Wahlkreis ausgewählt werden. Der Datensatz wird analog zur Darstellung aller Parteien über alle Wahlkreise hinweg abgebildet. Durch das Betätigen des Buttons können erneut die Stimmflüsse innerhalb einer Partei aus- und eingeblendet werden.
100 Mitglieder
Im Dropdown-Menu kann ein beliebiges Mitglied des Grossen Rates ausgewählt werden. Aus dem ensprechender Diagramm ist ersichtlich, woher das Mitglied seine Kandidatenstimmen veränderter Wahllisten erhalten hat.

Wahlsystem
Der Grosse Rat von Basel-Stadt wird jedes Schaltjahr für vier weitere Jahre gewählt. Die Wahl erfolgt in fünf Wahlkreisen nach dem Proporzverfahren. Wird eine Person auf einer Wahlliste aufgeführt, so erhält diese eine Kandidatenstimme und deren Partei eine Parteistimme.

Panaschieren
Es können Namen gestrichen und Kandidierende einer anderen Partei aufgeschrieben werden. Diese erhalten die Kandidatenstimmen und deren Partei die entsprechenden Parteistimmen.

Kumulieren
Der Name einer kandidierenden Person darf maximal dreimal aufgeführt werden.

Kandidaten- und Zusatzstimmen
Die Parteistärke errechnet sich aus Kandidaten- und Zusatzstimmen. Die Zusatzstimmen sind die nicht ausgefüllten, leeren Linien auf der Wahlliste einer Partei.

Freie Liste
Wird einem leeren Wahlzettel keine Liste zugewiesen, so verfallen die Zusatzstimmen.

Veränderte Wahllisten
Die veränderten Wahllisten umfassen die freie Liste sowie sämtliche vorgedruckten Listen, an welchen Veränderungen vorgenommen wurden.

Weitere Informationen finden sich im Wahlgesetz oder auf der Seite der Staatskanzlei des Kantons.
Spezialfall Bettingen
Der Wahlkreis Bettingen hat nur einen Sitz zugute. Da nur zwei Kandidaten zweier verschiedener Parteien zur Wahl standen, gab es in diesem Wahlkreis keine Stimmen veränderter Wahllisten.

Rohdaten

Die Daten stammen von der Fachstelle für OGD Basel-Stadt und können in Rohform hier heruntergeladen werden. Für die Visualisierung wurden die Daten aufbereitet.
